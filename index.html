Steps to work with git and github

First, Download git from https://git-scm.com/ and install it on your system.

Second, Create and account for the github https://github.com/ and login into your account.

In your github account create a repository with any name.

Creating a project with "VS Code-Git-Github" name

Create a file with named "index.html" or any other files that you need in your project

Open Activity bar, Click on Source Control(Ctrl + Shift + G)

Click on + on the right corner of this Activity bar.

Pick workspace folder to initialize git repo in "VS Code-Git-Github"

This will initialize git to your project and create a branch with name master. Please note that git is only upto your system. Now you need to upload it to github.

Make any change to the index.html file. Now you can commit it into the same activity bar. with a message of commit.
Now click on the vertical dots, to push your code to github.
Now you need to enter your email and password for the first last time the project. Enter it and proceed further. Now Copy repository path and paste it into terminal and press enter. Now again try it push your code on github.

Just changed the folder location

It will work as same before.

Creating branch using either click on right bottom on git master sign or click (ctrl + shift + p). Now right create branch

Enter name of the branch say "Version 1.0" | Please note this branch is created locally not to the github

Then enter the same path of your repository.

When you push your code to github, it will ask you to create a branch into your github account.

This will create a new branch to github and save your code on new branch

Go to your branch say version 1.0 into your github account.

Click on new pull request.

Fill the details of your branch or the new features added to it(something like that)

then click on the "create pull request".

It will check what new you have added to that branch so that it can be merged to master branch.
Click on the "merge". The whole code of that branch will be merged and updated. according to data that was in your branch.

Click one the fork

You can add or delete a file of another respository, the changes done will be stored in new branch of your github account.

You have to raise a pull request to owner of that project. If he likes your code. he will merge it with his own project.
